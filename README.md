# Teoría cuántica básica

_Este simulador contiene funciones para:_

- Calcular la probabilidad de que una particula se encuentre en una posicion en especifico
- Calcular la probabilidad de que un vector ket transite a otro vector
- Calcular la varianza y el valor propio de un observable en un estado dado
- Calcular los valores y vectores propios de un observable
- Calcular la probabilidad de que un sistema transite a alguno de los vectores propios despues de la observacion
